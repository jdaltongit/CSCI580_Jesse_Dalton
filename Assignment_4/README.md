# CSCI 580 Assignment 4 — TSP with Simulated Annealing and Genetic Algorithms

## How to Run

1. Open the notebook in Google Colab by clicking the .ipynb file in this repository and selecting "Open in Colab"
2. Run all cells in order from top to bottom using Runtime → Run All
3. All plots and outputs will appear inline below each cell after running

## What the Code Does

- Part 1-4: Sets up the TSP problem and baseline nearest neighbor solution
- Part 5A: Simulated Annealing implementation
- Part 5B: Genetic Algorithm implementation
- Part 6: Data collection, visualization and comparison of both algorithms

## Results

- Nearest Neighbor baseline: 6.0214
- SA best length: 5.0057 (16.87% improvement)
- GA best length: 5.3621 (10.95% improvement)
